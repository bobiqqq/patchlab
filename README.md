# PatchLab

Учебная Flask-платформа для исправления уязвимых мини-сервисов.
URL: <будет потом>

## Как это работает

Пользователь регистрируется, выбирает задачу, скачивает `vuln.py`, меняет код только между строками:

```python
### НАЧАЛО БЛОКА РЕДАКТИРОВАНИЯ ###
### КОНЕЦ БЛОКА РЕДАКТИРОВАНИЯ ###
```

Затем файл загружается обратно. Платформа собирает итоговый код и запускает checker задачи в отдельном процессе.

## Структура

- `app.py` — страницы, регистрация, вход.
- `api.py` — загрузка патча и JSON API.
- `models.py` — ORM-модели.
- `checker.py` — базовая проверка и ограниченный запуск патча.
- `tasks_methods.py` — загрузка задач и запуск checker-ов.
- `task_sources/` — задачи.

## Добавление задачи

Создать папку в `task_sources` и добавить:

- `info.json`
- `README.md`
- `vuln.py`
- `checker.py`
