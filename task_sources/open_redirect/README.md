# Защита redirect-параметра

Функция `is_safe_redirect` проверяет адрес, на который пользователь перейдёт после входа.

## Проблема

Сейчас функция принимает любой URL. Из-за этого злоумышленник может подсунуть внешний адрес и увести пользователя с сервиса.

## Цель

Разрешите только локальные относительные пути вроде `/profile`.
