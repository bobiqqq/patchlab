# Защита обновления профиля

Функция `update_profile` принимает текущего пользователя и данные формы.

## Проблема

Сейчас в профиль копируются все поля из `data`. Пользователь может передать `role=admin` и повысить себе права.

## Цель

Обновляйте только безопасные поля профиля, например `display_name` и `bio`.
