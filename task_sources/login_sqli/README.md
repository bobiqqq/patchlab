# Защита формы входа от SQL-инъекции

Функция `login` уязвима к SQL-инъекции, потому что собирает SQL-запрос через f-строку.

## Цель
Исправьте SQL-инъекцию с помощью параметризованного запроса.

## Пример атаки
Злоумышленник может войти как `admin` без пароля, если передаст `admin' --` в поле имени пользователя.
