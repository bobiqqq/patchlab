# Проверка прав доступа

Функция `can_open_admin_panel` решает, можно ли открыть админ-панель.

## Проблема

Сейчас доступ выдаётся по имени пользователя `admin`. Это слабая проверка: пользователь без роли администратора может получить доступ, если его username равен `admin`.

## Цель

Проверяйте роль пользователя, а не имя.
